# 832301309_concacts_backend

项目简介：

本项目为通讯录管理系统（Contacts Management System）的 后端部分，采用 Flask + SQLAlchemy + SQLite构建，实现用户注册、登录、联系人信息的增删改核心功能。  
后端通过 RESTful 风格的接口与前端进行数据交互，支持跨域访问，实现前后端分离的应用架构。
